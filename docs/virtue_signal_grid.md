# VIRTUE SIGNAL: Topic x Gameplay Grid

## 20 Topics x 7 Gameplay Types = 140 Mission Seeds

Every cell is a one-line game idea. Pick any cell, build it out. The best missions combine 2-3 cells from the same row (same topic, multiple gameplay types) or the same column (same gameplay, multiple topics layered).

---

## KEY

**CHASE** = Driving/chases, escaping police, tailing suspects, timed deliveries under CCTV pressure
**STEALTH** = Avoiding cameras, disguises, restricted zones, infiltration
**COMBAT** = Faction brawls, riot scenarios, police confrontations, defence
**PUZZLE** = Connecting evidence, hacking terminals, cracking codes, escape-room logic, detective work
**PHYSICAL** = Climbing, sabotage, planting devices, hoisting flags, parkour
**HEIST** = Multi-phase operations: plan, execute, improvise when it goes wrong
**SURVIVAL** = Hold position, defend a location, timed endurance, keep a signal alive

## HOW TO USE THIS GRID

1. **Pick a cell.** Any cell is a mission seed.
2. **Combine cells from the same row** for a multi-phase mission. E.g., GROOMING GANGS: PUZZLE (evidence board) + STEALTH (police station) + SURVIVAL (broadcast) = a three-phase investigation mission.
3. **Combine cells from the same column** for a mission that layers multiple topics. E.g., CHASE column: Ozempic delivery + knife crime ambulance run + housing eviction van = a "Night Shift" mission where the player does three emergency deliveries in one night, each from a different topic.
4. **Identify your favourite 28 cells** for the main mission arc.
5. **Use remaining cells** for side missions, optional content, and faction-specific jobs.

The grid is the menu. The missions are the meals.
